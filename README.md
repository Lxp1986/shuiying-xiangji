# 工程水印相机

给现场照片叠加「工程记录」风格水印的 Web 应用。样式对齐常见施工记录水印：

- 蓝色标题条（如「工程记录」）
- 黄色副标题条（项目名称 / 阶段）
- 半透明白底字段列表（施工内容、时间、天气、经纬度等）

## 功能

| 功能 | 说明 |
|------|------|
| 选图 / 拍照 | 支持相册、摄像头、拖放图片 |
| 水印大小 | 整体缩放框体 |
| 文字大小 | 独立调节字号 |
| 透明度 | 整体半透明，贴近实拍叠加效果 |
| 位置 | 九宫格锚点 + 可拖拽微调 |
| 类型模板 | 内置工程记录 / 安全巡查 / 进度汇报，可新建、编辑、删除 |
| 字段 | 可增删字段、改标签与内容；一键填时间 / 定位 |
| 导出 | 导出带水印的 JPEG |

## 使用

用浏览器直接打开即可：

```bash
# 方式一：双击 index.html

# 方式二：本地静态服务（推荐，定位等 API 更稳定）
cd "shuiying xiangji"
python3 -m http.server 5173
# 浏览器打开 http://localhost:5173
```

1. 点击「选择图片」或「拍照」
2. 在右侧「内容」填写标题、副标题与字段
3. 在「样式」调节大小、透明度、位置与颜色
4. 在「类型」管理水印模板
5. 点击「导出图片」保存

数据保存在浏览器 `localStorage`，刷新不会丢失自定义类型。

## 文件

- `index.html` — 页面结构
- `styles.css` — 界面样式
